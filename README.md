# Korrel

## Project Description
Korrel is a project designed to manage and organize various functionalities, including templates, static files, and Python scripts. It includes features for web-based metrics, switchboard management, and more.

## Project Structure

- **adc_temp.py, boot.py, logging.py, main.py, etc.**: Core Python scripts for the project.
- **static/**: Contains CSS, JavaScript, and other static assets.
  - `ADCmetrics.css`, `metrics.js`, `moment.min.js`, etc.
- **templates/**: HTML and Python template files for rendering web pages.
  - `index.html`, `switchboard.html`, etc.
- **utemplate/**: Template-related Python scripts.
  - `compiled.py`, `recompile.py`, etc.

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/korrel.git
   ```
2. Navigate to the project directory:
   ```bash
   cd korrel
   ```
3. Run the main script:
   ```bash
   python main.py
   ```
